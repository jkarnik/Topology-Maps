Key UI Features

L2 / L3 Toggle
Switch between physical cable view (L2) and logical subnet/VLAN view (L3) with one click
Drill-Down
Click any floor switch to see its 48 ports, connected APs, and endpoints. Breadcrumb navigation back up.
Edit Mode
Toggle edit mode to drag connections between ports. Changes propagate through the full SNMP loop.
Live Status
Green/red/amber indicators on every device and link. Metrics like throughput shown on hover.
