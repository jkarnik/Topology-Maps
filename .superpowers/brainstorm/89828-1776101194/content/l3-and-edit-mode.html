How the two views differ

L2 Physical View
  • Shows actual device-to-device cable connections
  • Hierarchical: Firewalls → Core → Floor → APs → Endpoints
  • Port-level detail on drill-down
  • Edit mode works here — rewire physical connections
L3 Logical View
  • Shows subnets, VLANs, and IP routing relationships
  • Grouped by network segment, not physical location
  • Click a VLAN to see its member devices
  • Shows inter-VLAN routing through the FortiGate
