## Key codebase facts for executor

- **Python tests:** `pytest` — run from repo root as `python -m pytest server/tests/<file> -v`
- **TypeScript check:** `cd ui && npx tsc --noEmit`
- **DB fixture pattern:** monkeypatch `database.DB_PATH` with a temp dir; call `database.get_connection()`
- **Store functions** all take `sqlite3.Connection` as first positional argument
- **FastAPI app** is at `server/main.py` — import as `from server.main import app`
- **Frontend API helper** is `_fetch<T>()` in `ui/src/api/config.ts`
- **Existing hooks** live in `ui/src/hooks/` — follow the pattern in `useConfigOrgs.ts`
- **No writes to Meraki APIs** — this is observability-only; diff engine is pure data transformation
